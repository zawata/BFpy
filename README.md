# BFpy
a BrainFuck Interpreter written in Python

# Implementation Details

    Left Cells             = Unbounded
    Right Cells            = Unbounded
    Cell Size              = 64b
    NewLine                = 0x10
    Right Brace Check      = True
    EOF                    = !
    Comments               = #Comment#
    Spaces, tabs, newlines = allowed
    Invalid Characters     = warn
